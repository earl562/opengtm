# Code of Conduct

This project is an open source effort. We aim for a low-drama, high-signal collaboration environment.

## Expected behavior

- Be respectful and direct.
- Assume good intent; ask clarifying questions.
- Keep discussions technical and focused on outcomes.
- Help others learn: share context, not just conclusions.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Threats or doxxing.
- Repeated disruptive behavior after being asked to stop.

## Reporting

If you experience or witness unacceptable behavior, report it privately to: security@opengtm.org.

## Enforcement

Maintainers may remove content, warn contributors, or restrict participation when necessary to protect the community.
