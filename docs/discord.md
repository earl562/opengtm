# Discord Gateway

The Discord gateway lives in `@opengtm/gateway-discord`.

## Setup
You need:
- `DISCORD_TOKEN`
- Discord application `clientId`
- Optional `guildId` for dev registration

## Commands
- `/opengtm help`
- `/opengtm status`
- `/opengtm init`
- `/opengtm run`
- `/opengtm approvals`

Approvals are rendered with Approve/Deny buttons.
